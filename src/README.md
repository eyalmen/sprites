
Documentation for flags used.

# sprites/gen1

- `-b` Back sprite
- `-g` Game it is from, can be `rb`, `rg`, `rgb`, or `y`

# models, dex, afd, previews

- `-b` Back sprite
- `-s` Shiny
- `-g` Gmax
- `-f` Female

# minisprites/pokemon

- `-v` Vendor is someone else than the official games, can be `smogon` (document `duplicate` and `g5`?)
